# FE Intern Task – Maska Landing Page

This project is a front-end assignment built using React + Vite.

## Objective
- Recreate a landing page inspired by https://face-oil.vercel.app/
- Use content from Maskabutters product page
- Design 4 creative Word of Mouth (WoM) / review sections

## Tech Stack
- React
- JavaScript
- CSS (no UI libraries)
- Vite

## Features
- Responsive landing page
- Product showcase with live social proof widgets
- **Four distinct review section variations:**
  - **Variation A - Ticker**: Minimal ticker with live counters + quick stats
  - **Variation B - Media**: Media-first UGC wall with image upload
  - **Variation C - Data**: Modern dashboard with analytics + review management
  - **Variation D - Carousel**: Featured reviews carousel + live social proof banner

## Review Section Features
- ✅ Live animated counters (orders, ratings, verified reviews)
- ✅ Social proof widgets and banners
- ✅ Star rating UI (interactive in form, visual in display)
- ✅ Search, filter, and sort functionality
- ✅ Image lightbox/modal viewer
- ✅ Helpful/like buttons (persisted in localStorage)
- ✅ Time formatting ("2 days ago")
- ✅ Rating distribution charts
- ✅ Review form with validation and character counter
- ✅ Success messages and empty states
- ✅ Verified badges and trust indicators
- ✅ Mobile responsive design
